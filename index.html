使用Python转换markdown为HTML

前言

markdown是技术博客主比较喜欢的书写格式。很多博客引擎都实现了markdown书写的文档转换成静态的html文件。
markdown类库是markdown转换html的Python版实现。

环境准备

  • 首先通过pip安装markdown模块：
pip install markdown

代码示例

import markdown

// markdown提供两个接口，一个允许转换字符串，一个进行文件的转换。
// 1. 对字符串的转换
html1 = markdown.markdown(your_text_string)

// 2. 对文件进行转换
html2 = markdown.markdown(input=source_file, output=target_file, [encoding])

编写调用脚本

为了使用方便，我写了一个简单的调用脚本converter.py:

import sys, getopt, markdown

def parse(argv):
    try:
        opts,vals = getopt.getopt(argv, "i:o:")
    except getopt.GetoptError:
        print('usage: converter.py -i <source_file> [-o <target_file>]')
        sys.exit(2)

    source_file = ''
    target_file = ''
    for opt, val in opts:
        if opt == '-i':
            source_file = val
        elif opt == '-o':
            target_file = val

    if source_file == '':
        print('usage: converter.py -i <source_file> [-o <target_file>]')
        sys.exit(2)

    if target_file == '':
        target_file = source_file+'.html'

    markdown.markdownFromFile(input=source_file, output=target_file)
    print('convert successfully.')

if __name__ == '__main__':
    parse(sys.argv[1:])
